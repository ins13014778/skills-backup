# 类Claw 智能体威胁矩阵（杀伤链视角）

> 来源: AISS 绿盟大模型安全智链社区（https://aiss.nsfocus.com/ → 类Claw 威胁矩阵），快照 2026-06-17。
> 与 `gaarm-risk-matrix.md` **正交**：GAARM 是风险分类法（5 域 × 3 阶段），本表是**智能体杀伤链**（6 生命周期阶段 × 威胁项 + severity），并提供**配套防御矩阵**。
> ⚠️ 编号 `AML.xxx` 是 AISS **自有 ATLAS 式编号**，**非**官方 MITRE ATLAS 技术号（官方为 `AML.T00xx`）。引用时标注"AISS 类Claw 编号"，需官方 ATLAS 映射另查 MITRE ATLAS。

## 三、与本 Skill 其他矩阵的关系

| 维度 | 用途 |
|------|------|
| `gaarm-risk-matrix.md`（173 风险）| 按"是什么风险"分类（域×阶段），做风险登记/覆盖率 |
| 本表（杀伤链 36 威胁 + 39 防御）| 按"攻击走到哪一步"编排（基础设施→…→影响），做攻击链推演 + 防御反推 |
| `testing-methodology.md §10`（OWASP）| LLM/ASI/WSTG 合规映射 |

**实战用法**：L1 攻击面识别后，用本表 6 阶段定位当前杀伤链位置 → 沿杀伤链向"影响"推演下一跳 → 对每个命中威胁项反查防御矩阵是否落地（L4 防御反推）。GAARM 风险名 ↔ 本表威胁项多为同义（如 A-1-006 沙箱逃逸 ↔ GAARM 容器逃逸；P-1-002 间接注入 ↔ GAARM.0040）。
